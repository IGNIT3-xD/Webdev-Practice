The total width of an element should be calculated like this: Total element width = width + left padding + right padding + left border + right.
The total height of an element should be calculated like this: Total element height = height + top padding + bottom padding + top border + bottom border
When you set the width/height of an element, the element often appears bigger than you have set (because the element's border and padding are added to the element's specified width/height) The box-sizing property allows us to include the padding and border in an element's total width and height.
